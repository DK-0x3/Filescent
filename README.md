- ### React + Vite
- ### Redux Toolkit + RTK Query
- ### I18 Next
- ### React Router

## Используемая версия Node.js `22.14.0`

```
nvm install v22.14.0
```

```
nvm use v22.14.0
```


Создайте файл `.env` в корне проекта
```
VITE_API_URL=http://localhost:1941
```
